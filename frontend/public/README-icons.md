# Иконки PWA

Для работы PWA необходимо создать следующие иконки:

1. **pwa-192x192.png** - иконка 192x192 пикселей
2. **pwa-512x512.png** - иконка 512x512 пикселей

## Рекомендации:

- Используйте простой логотип или иконку автомобиля/ремонта
- Фон должен быть прозрачным или однотонным
- Иконки должны быть четкими и читаемыми на разных размерах
- Для maskable иконок оставьте безопасную зону (padding) около 20% от краев

## Временное решение:

Пока иконки не созданы, можно использовать простые цветные квадраты или временные иконки.
После создания иконок поместите их в папку `public/`.
